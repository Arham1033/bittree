# 🌳 Bittree – Link-in-Bio Platform

Bittree is a modern **Link-in-Bio web application** inspired by Bitly's Linktree experience. It allows users to create a personalized page containing all of their important links in one place.

The project features a visually rich landing page with interactive animations, reusable components, responsive layouts, and a database-powered form that lets users claim their own tree and save their links.

---

### 🌐 Landing Page
- Modern Link-in-Bio homepage
- Smooth animations and transitions
- Interactive 3D-style cards
- Rotating card effects

- Responsive design
- Professional UI inspired by Bitly

# 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/bittree.git
```

### Navigate into the project

```bash
cd bittree
```

### Install dependencies

```bash
npm install
```

### Create a `.env.local` file

```env
MONGO_URI=your_mongodb_connection_string
```

### Start the development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
